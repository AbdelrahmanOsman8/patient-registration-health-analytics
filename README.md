# Patient Registration & Health Age Analytics System

## Overview
This Python project simulates a basic patient registration system for healthcare settings. It captures patient demographics, generates a username, calculates life statistics based on age, and provides preventive care recommendations.

## Objectives
- Register patient name and contact information
- Generate patient portal username and identify email domain
- Calculate life statistics (months, days, hours lived)
- Provide preventive care recommendations based on age category

## Features
- Simple EMR-style patient registration
- Life statistics calculations
- Age-based clinical category and preventive recommendations
- Input validation for email and age

## Technologies
- Python 3

## Healthcare Relevance
Patient registration is a core component of Electronic Medical Records (EMR). This project demonstrates fundamental health informatics concepts including preventive care guidance and basic analytics for patient management.

## Future Improvements
- Integrate with SQLite for persistent patient records
- Add contact info validation and phone number storage
- Include more health analytics (BMI, risk factors)
- Develop a GUI for hospital intake use
